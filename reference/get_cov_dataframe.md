# Retrieve the coverage dataframe.

Function to retrieve the coverage dataframe used to initialize the
object.

## Usage

``` r
get_cov_dataframe(x, verbose = T)
```

## Arguments

- x:

  a mvnmm object.

## Value

the coverage dataset used to fit the model.

## Examples

``` r
if (FALSE) get_cov_dataframe(x)
```
