# Extract the estimated mean parameters.

Returns a dataframe `KxT` with the estimated mean paramaters `mu_kt` per
clone `k` and dimension `t`.

## Usage

``` r
get_mean(x)
```

## Arguments

- x:

  a mvnmm object.

## Value

dataframe of mean parameters.

## Examples

``` r
if (FALSE) get_mean(x)
```
