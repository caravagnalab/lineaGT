# Function implemented to estimate the real number of clones in each cluster.

Function implemented to estimate the real number of clones in each
cluster.

## Usage

``` r
estimate_n_pops(x, highlight = c(), vcn = NULL)
```

## Arguments

- x:

  mvnmm object

- highlight:

  clusters to show

## Value

named array reporting for each cluster the estimated true number of
populations.
