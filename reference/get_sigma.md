# Extract the estimated variance parameters.

Returns a dataframe `KxT` with the estimated variance paramaters
`sigma_kt` per clone `k` and dimension `t`.

## Usage

``` r
get_sigma(x)
```

## Arguments

- x:

  a mvnmm object.

## Value

dataframe of variance parameters.

## Examples

``` r
if (FALSE) get_sigma(x)
```
